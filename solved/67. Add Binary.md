## Description

Given two binary strings, return their sum (also a binary string).

The input strings are both <strong>non-empty</strong> and contains only characters <code>1</code> orÂ <code>0</code>.

<strong>Example 1:</strong>

<strong>Input:</strong> a = "11", b = "1"  
<strong>Output:</strong> "100"

<strong>Example 2:</strong>

<strong>Input:</strong> a = "1010", b = "1011"  
<strong>Output:</strong> "10101"

## Python code

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        diff = len(a) - len(b)
        if diff < 0:
            a = "0" * -diff + a
        if diff > 0:
            b = "0" * diff + b
        carry, res, l = 0, "", len(a)-1
        while l >= 0:
            s = int(a[l]) + int(b[l]) + carry
            if s == 2:
                res = "0" + res
                carry = 1
            elif s == 3:
                res = "1" + res
                carry = 1
            else:
                res = str(s) + res
                carry = 0
            l -= 1
        if carry:
            res = str(carry) + res
        return res
```
